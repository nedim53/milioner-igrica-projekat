# 💰 Who Wants to Be a Millionaire – Web Quiz Game

A web-based version of the famous TV quiz game, built with HTML, CSS, JavaScript, and Bootstrap. Test your knowledge and climb your way to the virtual million!

Game PREVIEW: https://whowantstobeamilionaire.vercel.app/


## 🕹️ Features

- 🎯 15 multiple-choice questions
- ⏳ Countdown timer per question
- 💡 Lifelines (50:50, Ask the Audience, Phone a Friend)
- 📱 Responsive design – playable on desktop and mobile
- 🎉 Final result summary at the end

---

## 🔧 Technologies Used

- **HTML5** – Markup structure  
- **CSS3 / Bootstrap** – Styling and responsive layout  
- **JavaScript** – Game logic and interactivity
